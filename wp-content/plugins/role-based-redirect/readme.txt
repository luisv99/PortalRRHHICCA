Redirect users to specific page after login, based on their the role and also option for hide adminbar and restrict dashboard access.

== Description ==
User redirect to his desire page after login logout. And also hide admin bar for desire user's role and restrict dashboard access to user on role based.

== Installation ==
1. Upload `role-based-redirect` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. That\'s it you are ready to go :) .
